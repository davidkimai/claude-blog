# Documentation Best Practices

Create clear, maintainable documentation.

## When to Trigger

**Use when user mentions:**
- "Write documentation", "create README"
- "API docs", "technical writing"
- "Document this code", "add comments"
- "How to use" documentation
- "Getting started" guide

## Content Sources

Composio skills + ecosystem best practices

## Documentation Types

- **README** - Project overview
- **API Reference** - Function docs
- **Guides** - How-to articles
- **Architecture** - System design
- **Contributing** - For open source

## Best Practices

- Use markdown consistently
- Include code examples
- Keep it up to date
- Write for your audience
- Add troubleshooting sections

## Example Triggers

- "Write a README for this project"
- "Create API documentation"
- "Document this function"
- "How do I write good docs?"
- "Add usage examples"
